# 像素机甲对战

一个基于HTML5 Canvas的2D像素风格双人对战游戏，支持单机模式和AI对战。

## 游戏特色

- 🎮 **双人对战** - 本地双人对战模式
- 🤖 **AI对战** - 三种难度级别（简单/普通/困难）
- 🗺️ **10张地图** - 多样化的战场环境
- 🎨 **自定义颜色** - 玩家可以自定义机甲颜色
- 🌞 **主题切换** - 支持白天和黑夜两种主题
- 💥 **技能系统** - 三种独特技能（治疗、冲刺、爆发）
- 🎯 **可破坏掩体** - 动态地图环境
- 🔊 **完整音效** - 射击、爆炸、技能音效和背景音乐
- 📱 **响应式设计** - 适应各种屏幕尺寸

## 游戏机制

### 技能系统

#### 治疗技能 (Skill 1)
- 恢复60点生命值
- 冷却时间：5秒
- 在血量低时使用效果最佳

#### 冲刺技能 (Skill 2)
- 瞬间移动一段距离
- 冷却时间：3秒
- **冲刺距离：300像素**（可穿越宽达250像素的掩体）
- 可以穿越障碍
- 用于追击或逃生
- 如果掩体宽度超过冲刺距离，会停在掩体边缘

#### 爆发技能 (Skill 3)
- 对周围敌人造成30点范围伤害
- 冷却时间：4秒
- 在近距离使用最有效

### 战斗系统

#### 近战攻击
- 伤害：15点
- 攻击距离：短
- 冷却时间：0.5秒

#### 远程攻击
- 伤害：10点
- 攻击距离：长
- 冷却时间：0.33秒

#### 防御机制
- 减少80%受到的伤害
- 防御时无法攻击
- 可以随时开始/停止防御

### 掩体系统
- 地图中心对称生成掩体
- 掩体可被子弹和近战攻击破坏
- **百分比伤害系统**：
  - 远程射击：每次破坏10%
  - 近战攻击：每次破坏20%
  - 爆发技能：每次破坏50%
- 显示血量状态（完整/受损/严重受损）
- 可以阻挡子弹，提供战术掩护

## 游戏模式

### 双人对战模式
- 两个玩家在同一设备上对战
- 公平的出生点和掩体布局
- 可先选择地图再开始战斗
- 先将对方血量降至零者获胜

### 单人模式 vs AI
- 与AI机器人对战
- 三种难度选择：
  - **简单** - 反应慢，射击精度低，不会使用技能，适合新手入门
  - **普通** - 平衡的对手，有一定挑战性
  - **困难** - 高反应速度和精准射击，频繁使用技能

## 自定义设置

### 颜色自定义
- 可以选择玩家1和玩家2的机甲颜色
- 颜色选择后自动保存
- 包括机甲本体、子弹和特效的颜色

### 血量上限设置
- 可调整范围：50-500
- 调整双方的最大生命值
- 影响游戏节奏和时长

### 主题切换
- **白天模式** - 明亮的游戏环境
- **黑夜模式** - 深色背景，更有氛围

### 音效设置
- **音效开关** - 开启/关闭所有音效
- **主音量** - 控制整体音量
- **音效音量** - 控制射击、爆炸等音效
- **音乐音量** - 控制背景音乐
- 所有设置自动保存

## AI系统

### 智能行为状态机

AI具有多种行为状态：
- **攻击** - 在理想射程内射击
- **接近** - 边移动边射击
- **追击** - 快速接近目标
- **撤退** - 后退同时反击
- **躲避** - 紧急回避威胁
- **防御** - 举盾防御

### 威胁检测

AI能够：
- 预测子弹轨迹，判断是否会被命中
- 检测近战攻击范围
- 根据威胁优先级做出反应

### 难度差异

| 属性 | 简单 | 普通 | 困难 |
|------|------|------|------|
| 反应速度 | 很慢 | 中等 | 极快 |
| 射击精度 | 40% | 65% | 95% |
| 躲避概率 | 5% | 20% | 60% |
| 技能使用 | 不使用 | 30% | 80% |
| 移动预判 | 无 | 无 | 有 |
| 攻击频率 | 40% | 60% | 100% |

## 如何运行

1. 下载项目到本地
2. 使用浏览器打开 `index.html` 文件
3. 选择游戏模式开始游玩！

> 注意：建议使用现代浏览器（Chrome/Firefox/Edge）获得最佳体验

## 胜利条件

- 成功将对手血量降低到零
- 胜利方显示王冠动画
- 失败方倒在地上
- 3秒后显示游戏结束画面
- 可选择：
  - **再来一局** - 在同地图重新开始
  - **换个地图** - 选择新地图
  - **返回主页** - 回到主菜单
